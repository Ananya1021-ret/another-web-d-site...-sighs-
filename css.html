CSS Guide

Types:

  1. Inline Style:
    Example:
    <p style="color:red;">Red text</p>
  2. Internal Style:
    Written in the head.
    Example:
    <style>
    body {
      background-color: #333;
      color: whitesmoke;
    }
    </style>
  3. External CSS:
    Best way.
    Example:
    <link rel="stylesheet" href="main.css">
