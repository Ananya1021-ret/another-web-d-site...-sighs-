What is HTML?

HTML or HyperText Markup Language is the backbone os webpages. It structures the contents you see online. Think of HTML as a skeleton. A website with only html looks like this one.

Basic Structures of an HTML Page

Every HTML document starts with a <!DOCTYPE html> and has two main parts: <head> and <body>

        
          <!DOCTYPE html>
          <html>
           <head>
            <title>My First Page</title>
           </head>
           
           <body>
             <!--Content that is visible on the webpage goes here-->
           </body>
         </html>

The body can be further divided into 3 parts: <header>, <main>, <footer>
Each page can have only one Header and Footer.
Header generally contents the title of the page and the table of contents or navigation.

Headings, Paragraphs, Lines

  • <h1> to <h6> : Heading tags, where <h1> is the largest, and <h6> is the smallest
  • <p>: for paragraph
  • <br>: for line breaks
  • <hr>: for horizontal line
Examples:
        
          <h1>Main Heading</h1>
          <h2>Subheading</h2>
          <p>This is a paragraph.</p>

Abbreviations & Address

<abbr> : Shows the full form when hovered over
<address> : Shows the address.

Examples:
        
          
          <abbr title="Mozilla Developer Network"
          href="https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Structuring_content">
          MDN
          </abbr>
          
          <address>
           MIT, Manipal
           Block 8, Room 416
           567104
          <address>

  • <ul> - Unordered List

Example:

         <dl>

Links and Navigation

  • <a>: Anchor tag for linking
  • <href>: Destination to where it is linking. Can be of two types: Absolute Reference (when referencing something that is not present in the folder), and Relative Reference.
  • target="_blank": attributes that opens link in new tab
  • download, mailto, tel: are other useful attributes
Examples:
  • <a href="https://mdn.com" target="_blank">MDN</a>
  • <a href="#section-id">Jump to section</a>
  • <a href="myimage.jpg" download>Download this</a>
  • <a href="mailto:me@example.com">Email Me</a>
  • <a href="tel:+911234567890">Call Me</a>
Navigation:
        
          <nav>
           <ul>
            <li><a href="#home">Home</a></li>
            <li><a href="page.html">Page</a></li>
           </ul>
          </nav>

Comments

HTML comments are ignored by the browser. These help you explain your code, temporarily disable parts, or keep some notes for your own

Example:
<!--comment-->

Images

Displays an image on page

  • src: source path, contains the source/link of the image
  • alt: alternative text, shows when the image doesn't load
  • loading="lazy": loads image only when needed. Enchances the performance
  • title, width, height: these can also be added as attributes
Example:
        
          <figure>
            <img src="image.jpg" alt="A blue image" loading="lazy">
            <figcaption>Blue Period</figcaption>
          </figure>

Semantic Tags

Semantic tags descrive the purpose of the content

  • <header> Header </header>
  • <main> Main Content </main>
  • <footer> Footer </footer>
  • <article> An article </article>
  • <section> Group of content </section>
  • <aside> Sidebar/extra info </aside>
  • <nav> Navigation links </nav>
for toggle:
        <details>
          <summary>Click to open</summary>
          <p>This is hidden content.</p>
        </details>

Forms:

to collect user input:

  1. Basic Form:
    
              <form action="https://httpbin.org/get" method="get">
               <label for="name">Name:</label>
               <input type="text" id="name" name="name" required>
              </form>
  2. Input Fields:

    • <input type="password" required>
    • <input type="number" min="1950" max="2020" step="10">
    • <input type="tel" pattern="[0-9]{5}[0-9]{5}">

  3. Select or Datalist:
    
                <select>
                  <option>Milk Coffee</option>
                </select>

                <input list="coffee">
                <datalist id="coffee">
                  <option value="Black Coffee">
                </datalist>
  4. Radio Buttons and Checkboxes:
    
              <input type="radio" name="food" value="chicken"> Chicken  
              <input type="checkbox" name="pets" value="dog"> Dog
  5. Textarea and Buttons:
    
              <textarea rows="5" cols="30"></textarea>
              <button type="submit">Submit</button>
              <button type="reset">Reset</button>
